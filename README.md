# Modbus_client
## Инструкция по запуску
1. Установите программу com0com
2. В программе необходимо создать пару портов вида COMx ↔ COMy
3. Запустить консольный симулятор Modbus‑устройства. В консоли ввести COMx
4. Скачать архив с приложением, разархивировать его в любую папку на компьютере
5. В папке найти файл Modbus_client.exe и запустить его
6. На вкладке "Подключение" ввести COMy в поле порта и нажать кнопку "Подключить". Появится сообщение "Подключено к COMy".

## Описание структуры приложения
Приложение разработано на WPF с использованием архитектуры MVVM и состоит из следующих файлов:
- ModbusService.cs - это слой работы с устройством. Подключение к порту, чтение регистров, запись значения в 12 регистр. Все операции обёрнуты в try/catch
- ViewModel.cs - это связь между интерфейсом и ModbusService. Обеспечивает привязку данных между интерфейсом и ModbusService, управляет состоянием подключения, запускает polling регистров с интервалом 1 секунда и ведёт журнал событий
- Register.cs - это строки таблицы регистров. Содержит адрес и значение регистра
- RelayCommand.cs - это вспомогательный класс для реализации команд для MVVM
- MainWindow.xaml - это пользовательский интерфейс с тремя вкладками: подключение, регистры и журнал событий
